# Calculadora de IMC - Projeto Web --> Deploy --> https://calculadoraimcmedical.netlify.app

## Descrição

Este projeto consiste em uma **Calculadora de IMC** (Índice de Massa Corporal) desenvolvida com HTML, CSS e JavaScript, que permite ao usuário calcular seu IMC com base no peso e altura inseridos. A aplicação é responsiva e conta com uma interface limpa e moderna, ideal para dispositivos móveis e desktop.

## Funcionalidades

- **Cálculo de IMC**: O usuário insere seu nome, peso e altura, e a calculadora retorna o valor do IMC e a classificação (abaixo do peso, peso ideal, sobrepeso, obesidade grau I, II, ou III).
- **Interface Responsiva**: O design foi desenvolvido para se adaptar a diferentes tamanhos de tela, proporcionando uma boa experiência tanto em dispositivos móveis quanto em desktop.
- **Efeito de Desfoque**: Após calcular o IMC, o resultado é exibido em uma caixa centralizada com o fundo desfocado, destacando o resultado.
- **Botão de Fechar**: O resultado pode ser fechado, e todos os campos são limpos automaticamente ao clicar em "Fechar".

## Tecnologias Utilizadas

- **HTML5**: Estruturação do conteúdo da página, incluindo formulários, botões e exibição de resultados.
- **CSS3**: Estilização da página, garantindo um design responsivo, com efeitos de transição suave, como o desfoque do fundo ao mostrar o resultado.
- **JavaScript**: Implementação da lógica de cálculo do IMC e manipulação de DOM para exibir os resultados.

## Como Usar

1. Clone este repositório para o seu computador:
   ```bash
   git clone https://github.com/seu_usuario/calculadora-imc.git
2.Abra o arquivo index.html em seu navegador de preferência.

3.Insira o seu nome, peso e altura, e clique em "Calcular". O seu IMC será exibido junto com a classificação correspondente.

4.Para refazer o cálculo, basta fechar o resultado e preencher os campos novamente.
